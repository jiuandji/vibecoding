# 💡 Идея

Планы следующих шагов, вырастающие из выводов `иследование/`.

## Содержание

- [`practice-4.md`](./practice-4.md) — основная концепция: **Cosmos Archive** (Three.js + deep space тема)
- [`альтернативы.md`](./альтернативы.md) — три других направления (SPA, Shader Art, Portfolio)

## Логика выбора

В `иследование/` выявлены две оси прогрессии:
1. **Анимационная:** Canvas → Canvas ООП → GSAP → *Three.js*
2. **Визуальная:** Light → Warm dark → OLED → *Deep space*

Основная идея `practice-4` продолжает обе оси одновременно — это естественное продолжение изученного пути.
