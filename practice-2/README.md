# 🍕 Pizza Fuoco — Practice 2

Лендинг пиццерии с огненной тематикой и тёмным дизайном.

## Превью

Тёмная тема · Огоньки · Оранжево-золотые акценты · Canvas-частицы

## Стек

- HTML5 (семантическая разметка)
- CSS3 (Grid, Flexbox, градиенты, анимации)
- Vanilla JavaScript (Canvas API, OOP, IntersectionObserver)
- Google Fonts: Playfair Display + Inter

## Секции

1. 🔥 Hero — центрированный с animated title + 3 CTA
2. 🍕 Меню — классика, фирменные, напитки (с описаниями)
3. 📞 Контакты — 4 карточки-иконки
4. 🎉 Галерея праздников — 6 фото с hover overlay
5. 🕐 Часы работы — таблица
6. 📍 Футер

## Фишки

- Поднимающиеся огоньки (Canvas + класс Ember)
- Floating анимация логотипа
- Hover overlay на галерее
- Scroll-анимации с staggered delay
- 3 цветных CTA-кнопки (синяя, оранжевая, зелёная)

## Запуск

Открыть `index.html` в браузере.
