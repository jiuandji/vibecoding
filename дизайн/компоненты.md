# 🧩 UI-компоненты

Спецификация переиспользуемых компонентов для practice-4. Опирается на токены из `визуальный-язык.md`.

## 1. Навигация (Nav)

### Desktop
- Fixed top, высота 72px
- Transparent изначально, при скролле > 50px: `backdrop-filter: blur(12px)` + `background: rgba(5, 7, 22, 0.8)`
- Логотип слева (text + маленькая 3D-точка рядом)
- Ссылки справа: `font-family: mono`, `uppercase`, `letter-spacing: 0.1em`
- Активная ссылка: нижняя линия `1px solid --color-nebula` + `--glow-nebula`

### Mobile (≤ 768px)
- Burger-иконка справа (3 линии)
- Клик → fullscreen overlay
- Overlay: `background: var(--color-void)`, ссылки крупно по центру
- Закрытие — крестик или повторный тап по burger

## 2. Кнопки

### 2.1 Primary CTA
```
height:      48px
padding:     0 24px
background:  var(--color-nebula)
color:       var(--color-starlight)
border:      none
border-radius: var(--radius-sm)
font:        mono, 500, uppercase, letter-spacing 0.1em
box-shadow:  var(--glow-nebula)
```

**States:**
- Hover: масштаб 1.02, glow интенсивнее (opacity 0.6)
- Active: масштаб 0.98
- Magnetic effect (из practice-3): GSAP elastic.out, amplitude 12px

### 2.2 Secondary (Ghost)
```
height:        48px
padding:       0 24px
background:    transparent
color:         var(--color-starlight)
border:        1px solid var(--color-horizon)
font:          mono, 500, uppercase
```

**Hover:** border-color → `--color-nebula`, glow soft

### 2.3 Icon Button
- 40x40px
- Background прозрачный
- Иконка 20x20 цвета `--color-dust`
- Hover: цвет → `--color-starlight`, background → `rgba(107, 92, 255, 0.08)`

## 3. Карточки

### 3.1 Mission Card (3D-превью миссии)

```
aspect-ratio: 3 / 4
background:   var(--color-abyss)
border:       var(--border-subtle)
padding:      var(--space-6)
overflow:     hidden
```

**Структура:**
1. 3D-превью зонда (верхняя половина, Three.js canvas)
2. Тег миссии (mono, uppercase, цвет dust)
3. Название (Space Grotesk 600, H3-размер)
4. Год / статус (mono, small, цвет dust)

**Hover:**
- Border → `--color-nebula`
- 3D-модель начинает вращаться (если не крутилась)
- Glow: `--glow-nebula` с opacity 0.3

### 3.2 Data Card (статистика)

```
padding:    var(--space-8) var(--space-6)
text-align: center
border:     var(--border-subtle) (только top и bottom)
```

**Содержимое:**
1. Число (mono, display-size, цвет starlight)
2. Единица измерения (mono, small, цвет nebula, uppercase)
3. Подпись (body, цвет dust)

**Анимация:** счётчик от 0 при входе в viewport (`CounterAnimator`).

## 4. Формы

### 4.1 Input (text / email)
```
height:         56px
padding:        0 var(--space-4)
background:     transparent
border:         none
border-bottom:  1px solid var(--color-horizon)
color:          var(--color-starlight)
font:           Space Grotesk, 400, body-size
transition:     border-color 0.3s ease
```

**States:**
- Focus: `border-bottom-color: var(--color-nebula)` + subtle glow
- Error: `border-bottom-color: var(--color-plasma)` + маленький текст ошибки снизу
- Valid: лёгкий чекмарк справа (mono glyph)

### 4.2 Label
- Floating label pattern: сначала внутри inputа как placeholder, при focus/value — поднимается наверх и уменьшается
- Font: mono, uppercase, letter-spacing 0.1em, small-size, цвет dust

### 4.3 Textarea
- Аналогично input, но `min-height: 120px` и `resize: vertical`
- Счётчик символов снизу (если есть лимит): mono, small, цвет dust

### 4.4 Select
- Стилизованный через кастомный dropdown (не native)
- Trigger выглядит как input
- Options: overlay с `background: var(--color-abyss)`, `border: var(--border-subtle)`

### 4.5 Submit button
- Использует Primary CTA
- После submit: кнопка трансформируется в success-индикатор (mono-текст "TRANSMITTED" с чекмарком)

## 5. Custom Cursor

Наследуется из practice-3, но переосмыслен как **прицел/визир**:

### Default state
- Внешнее кольцо 32x32, `border: 1px solid var(--color-starlight)`, opacity 0.6
- Центральная точка 4x4, цвет `--color-nebula`
- Плавное следование с lerp (0.15)

### Hover на интерактивном элементе
- Кольцо увеличивается до 48x48
- Цвет кольца → `--color-nebula`
- Glow добавляется

### Click state
- Кольцо сжимается до 24x24 на момент клика
- Возврат через `elastic.out` (GSAP)

**На touch-устройствах:** полностью скрыт, возвращается системный курсор/тап.

## 6. Timeline

### Структура
- Горизонтальная ось, идущая через всю секцию
- Узловые точки (mission markers): 12x12 кружки с glow
- Год под каждой точкой (mono, small)
- Название миссии над точкой (Space Grotesk 500, small)

### Интерактивность
- Скролл страницы → камера 3D-сцены движется вдоль оси Z
- Точки, которые "проходят мимо камеры", подсвечиваются

## 7. Page Loader

Наследуется из practice-3, адаптирован под тему:

- Fullscreen overlay `background: var(--color-void)`
- По центру: пульсирующая точка (1 звезда) + текст "INITIALIZING..." моноширинным
- Progress bar снизу (1px, растущая линия `--color-nebula`)
- Скрывается плавным fade-out после инициализации Three.js (минимум 800ms для ощущения значимости)
