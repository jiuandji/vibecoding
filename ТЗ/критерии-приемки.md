# ✅ Критерии приёмки

Чеклист, по которому `practice-4` считается готовым. Соответствует требованиям `ТЗ/тз.md`.

## Визуальные

- [ ] Все 6 секций из ТЗ присутствуют и сверстаны
- [ ] Палитра соответствует ТЗ (void/nebula/plasma/starlight/dust)
- [ ] Шрифты Space Grotesk + JetBrains Mono подгружены
- [ ] Дизайн консистентен с descriptionом "deep space / tech aesthetic"
- [ ] На мобильных (≤640px) — нет горизонтального скролла
- [ ] На десктопе (≥1025px) — полная раскладка без разрывов

## Функциональные

- [ ] Starfield рендерится и анимируется
- [ ] Параллакс от мыши работает (заметное, но не агрессивное смещение)
- [ ] Custom cursor появляется на десктопе, скрыт на touch-устройствах
- [ ] Magnetic CTA-кнопки реагируют на курсор
- [ ] Карточки миссий реагируют на hover (вращение или другой эффект)
- [ ] Timeline реагирует на скролл (движение камеры или другой визуальный отклик)
- [ ] Счётчики анимируются при появлении в viewport
- [ ] Форма валидируется (пустые поля, email-формат)
- [ ] Форма показывает success-state после отправки
- [ ] Page loader скрывается после инициализации Three.js

## Технические

- [ ] FPS ≥ 60 на десктопе (Chrome DevTools Performance)
- [ ] FPS ≥ 30 на мобильных (реальное устройство или эмуляция)
- [ ] Lighthouse Performance ≥ 80
- [ ] Консоль без ошибок
- [ ] Консоль без warnings
- [ ] JS-бандл ≤ 800KB
- [ ] Первый FCP < 2 сек на эмуляции 4G

## Архитектурные

- [ ] Минимум 6 JS-классов (продолжение линии practice-3)
- [ ] Используются CSS Custom Properties для всей палитры и типографики
- [ ] Три.js импортируется корректно (не загружается дважды)
- [ ] GSAP используется для UI-анимаций (не переизобретаются magnetic buttons)
- [ ] Нет jQuery, нет фреймворков — только ванильный JS

## Доступность / SEO

- [ ] Все изображения имеют `alt`
- [ ] Семантические теги: `<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`
- [ ] Фокус-состояния видимы на всех интерактивных элементах
- [ ] `<title>`, `<meta description>`, `<meta og:*>` заполнены
- [ ] Форма имеет `<label>` для каждого `<input>`
- [ ] Контраст текста и фона проходит WCAG AA

## Устойчивость

- [ ] Если WebGL недоступен — показывается статичный fallback-фон
- [ ] Если JS отключен — базовая структура и текст читаемы
- [ ] Ресайз окна не ломает 3D-сцену
- [ ] На iPhone / Android смартфоне ничего не мерцает

## Документация

- [ ] В `practice-4/` создан `README.md` по образцу `practice-3/README.md`
- [ ] Корневой `README.md` обновлён — добавлен блок про practice-4
- [ ] Структура проекта в корневом `README.md` обновлена

## Financial / scope check

- [ ] НЕ написан бэкенд (out of scope)
- [ ] НЕ добавлены лишние страницы (только одностраничный лендинг)
- [ ] НЕ внедрены фреймворки

3. Корневой `README.md` обновлён
