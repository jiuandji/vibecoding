# 🌸 Barista Loco — Practice 1

Лендинг кофейни в Мадриде с японской эстетикой цветущей сакуры.

## Превью

Светлая, тёплая тема · Сакура · Кремовые тона · Canvas-лепестки

## Стек

- HTML5 (семантическая разметка)
- CSS3 (Grid, Flexbox, анимации)
- Vanilla JavaScript (Canvas API, IntersectionObserver)
- Google Fonts: Playfair Display + Inter

## Секции

1. 🌸 Hero — split-layout с изображением
2. ☕ Меню — кофе, десерты, напитки (с ценами)
3. 📸 Галерея — карусель с 6 фото + свайп
4. 📖 О нас — текст + статистика
5. 📅 Бронирование — форма с валидацией
6. ⭐ Отзывы — 3 карточки
7. 📍 Футер / Контакты

## Фишки

- Падающие лепестки сакуры (Canvas 2D)
- Карусель с автоплеем, стрелками и точками
- Touch/swipe поддержка
- Мобильное меню
- Плавающая CTA-кнопка
- Success-состояние формы

## Запуск

Открыть `index.html` в браузере.
