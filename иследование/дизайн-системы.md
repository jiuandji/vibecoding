# 🎨 Исследование дизайн-систем

Сравнение дизайн-языков трёх лендингов и выводы о принципах построения.

## Три эстетики

### 🌸 practice-1 — "Мягкая азиатская"
- **Настроение:** уютное, спокойное, женственное
- **Форма:** rounded corners (`border-radius: 12-24px`), мягкие тени
- **Цвет:** пастельная палитра, высокая светлота
- **Типографика:** Playfair Display (serif, элегантный) + Inter (sans, читабельный)
- **Метафора:** сакура, кремовая пенка кофе, утренний свет

### 🍕 practice-2 — "Тёплая ресторанная"
- **Настроение:** уютное, но насыщенное, аппетитное
- **Форма:** rounded, но менее скруглённые, драматичные тени
- **Цвет:** тёмная база + тёплые акценты (огонь, золото)
- **Типографика:** та же пара (Playfair + Inter) — переиспользование
- **Метафора:** огонь печи, золотая корочка, вечерний свет

### ⌚ practice-3 — "Швейцарская роскошь"
- **Настроение:** сдержанное, премиальное, аристократичное
- **Форма:** **острые углы**, минимум скруглений
- **Цвет:** почти монохром — чёрный + приглушённое золото
- **Типографика:** Cormorant Garamond (более утончённый serif) + Inter
- **Метафора:** механизм часов, ночь, precision

## Закономерности

### 1. Типографическая пара работает везде
Inter как "рабочий" шрифт + серифный для заголовков — надёжная комбинация, которая адаптируется под любую тему.

### 2. Эстетика = форма + палитра + шрифт
Одни и те же секции (hero, галерея, футер) воспринимаются совершенно по-разному благодаря смене трёх параметров. Структура почти одинаковая — меняется обёртка.

### 3. Контраст растёт с уровнем "премиальности"
Чем дороже должен казаться продукт, тем сильнее контраст фона и акцента. Кофейня — мягкая, часы — максимально дихотомичные.

### 4. Border-radius как индикатор "теплоты"
- 24px → максимально дружелюбно (кофейня)
- 12-16px → уютно, но серьёзно (пиццерия)
- 0-4px → формально, премиально (часы)

## CSS-переменные как основа

Во всех проектах дизайн-система реализована через CSS custom properties:

```css
:root {
  --color-bg: ...;
  --color-accent: ...;
  --font-heading: ...;
  --radius-card: ...;
}
```

В `practice-3` это расширено до easing-переменных — шаг к полноценному design token system.

## Выводы для следующих проектов

1. **Начинай с трёх параметров:** тема (light/dark/oled), палитра (3-5 цветов), шрифтовая пара.
2. **Border-radius задаёт характер** сильнее, чем кажется.
3. **CSS-переменные — не просто удобство**, это основа масштабирования дизайн-системы.
4. **Переиспользуемые шрифты** (Inter) экономят когнитивную нагрузку и загрузку страницы.
