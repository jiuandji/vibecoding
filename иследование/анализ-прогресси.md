# 📈 Анализ прогрессии сложности

Исследование того, как усложняются технические задачи от `practice-1` к `practice-3`.

## Метрики роста

| Параметр | practice-1 | practice-2 | practice-3 |
|----------|-----------|-----------|-----------|
| Строк кода | 1608 | 931 | 1857 |
| CSS строк | 1001 | 622 | 1238 |
| JS строк | 252 | 105 | 331 |
| Классов в JS | 0 | 1 (`Ember`) | 5 |
| Внешних библиотек | 0 | 0 | 1 (GSAP) |
| CSS-переменных | базовый набор | базовый набор | расширенный (+easing) |

Интересно, что `practice-2` **короче** первого — это не регресс, а смена фокуса: меньше секций, но появилось ООП и работа с физикой частиц.

## Три оси усложнения

### 1. Визуальная ось: свет → тьма → OLED
- **practice-1:** тёплая светлая палитра, мягкие тени, rounded corners
- **practice-2:** тёмно-коричневая база, огненные акценты, радиальные градиенты
- **practice-3:** абсолютный чёрный (`#0a0a0a`), острые углы, минимализм

### 2. Анимационная ось: Canvas → Canvas+ООП → GSAP+классы
- **practice-1:** процедурный Canvas (лепестки падают)
- **practice-2:** ООП Canvas (класс `Ember`, физика частиц)
- **practice-3:** GSAP + кастомные классы (Text Scramble, Magnetic Buttons, Custom Cursor)

### 3. UX-ось: статика → скролл-эффекты → иммерсив
- **practice-1:** скролл + карусель + форма
- **practice-2:** staggered scroll-анимации через IntersectionObserver
- **practice-3:** page loader, следящий курсор, magnetic elements, film grain

## Что осваивается на каждом шаге

```
practice-1  →  practice-2  →  practice-3
   │              │               │
   ├─ Canvas 2D   ├─ JS Classes   ├─ GSAP library
   ├─ Grid/Flex   ├─ Dark UI      ├─ RequestAnimationFrame
   ├─ Carousel    ├─ Radial grad  ├─ Custom cursor (lerp)
   ├─ Forms       ├─ Aspect-ratio ├─ Text manipulation (spans)
   └─ Media query └─ Staggering   └─ mix-blend-mode, clamp()
```

## Выводы

1. Прогрессия построена **не по количеству**, а по качеству техник.
2. Каждый следующий проект добавляет **1 новый ментальный инструмент** (ООП → библиотека).
3. Дизайн всегда завязан на тему: сакура → огонь → luxury — это делает обучение эмоционально цельным.
